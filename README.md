# Losers — Community Website

A modern, responsive website for the **Losers** community — a vibrant social group built on friendship, shared interests, and togetherness.

## Pages

| Page                   | Description                                                     |
| ---------------------- | --------------------------------------------------------------- |
| `index.html`           | Home page with hero, about, leadership preview, events, and CTA |
| `about.html`           | Community story, mission, vision, values, and milestones        |
| `leadership.html`      | President, Secretary, Executive Members, and Branch Presidents  |
| `sub-communities.html` | Main branch and sub-branches (eFootball, COC)                   |
| `events.html`          | Upcoming and past community events                              |
| `blog.html`            | Community blog posts, announcements, and stories                |
| `contact.html`         | Contact form and email                                          |
| `efootball.html`       | eFootball branch page with members and stats                    |
| `coc.html`             | Clash of Clans branch page with members and stats               |
| `join.html`            | Membership application form                                     |

## Tech Stack

- **HTML5** — Semantic markup
- **CSS3** — Custom properties, Grid, Flexbox, animations, dark mode
- **JavaScript** — Vanilla JS for navigation, theme toggle, filtering, and form handling
- **Fonts** — Inter + Poppins (Google Fonts)
- **Icons** — Font Awesome 6.5

## Features

- Fully responsive design (mobile, tablet, desktop)
- Dark mode toggle with smooth transitions
- Scroll animations (fade-in on intersection)
- Event and blog filtering by category
- Contact form via mailto
- Consistent navbar and footer across all pages

## Getting Started

Simply open `index.html` in a browser — no build tools required.
